# 红岩大作业
## websocket框架
### 实现功能
#### ·升级链接
#### ·读取消息
#### ·写入消息
#### ·关闭链接（可指定状态码和错误信息）
#### ·设置读写的超时时间
#### ·可以设置读取和写入的最大字节数
#### ·ping pong心跳消息处理
#### ·可以处理消息分片
## Gin框架
### 实现功能
#### 路由管理（可分组，但是没写通配符)
#### 中间件
#### 上下文(封装了一些方法)
#### 没啥特别的，然后存储用的是和Gin一样的基数树(低配版)
